The Flexible Software Package (FSP) is Renesas's next-generation software platform for the RA MCU family. FSP provides best-in-class HAL drivers, RTOS integration, and portable middleware stacks for building secure, connected embedded applications.

## What is FSP?

FSP enables rapid development on Renesas RA microcontrollers by providing:

- **HAL drivers** with a consistent, interface-based API across all RA MCU variants
- **Middleware stacks** for networking (LwIP, NetX Duo), security (MbedTLS, RSIP), file systems (LittleFS, FileX), and more
- **RTOS integration** with FreeRTOS and Eclipse ThreadX (formerly Azure RTOS)
- **Motor control** middleware for vector control, sensorless, and encoder-based motor systems
- **e² studio IDE** with RASC configurator for visual peripheral setup and code generation

## Get started

<Steps>
  <Step title="Install e² studio with FSP">
    Download the FSP + e² studio installer from the [latest release](https://github.com/renesas/fsp/releases/tag/v6.4.0). This bundles the IDE, FSP packs, GCC/LLVM toolchains, and all required tools.
  </Step>
  <Step title="Create a new RA project">
    Open e² studio, go to **File → New → C/C++ Project**, and select **Renesas RA** in the left pane. The RASC configurator will open for visual peripheral configuration.
  </Step>
  <Step title="Add FSP modules">
    In the RASC **Stacks** tab, add the HAL drivers or middleware modules you need. RASC generates initialization code and pin configurations automatically.
  </Step>
  <Step title="Build and debug">
    Compile your project with the built-in GCC or LLVM toolchain and flash it to your RA kit using the integrated J-Link debugger support.
  </Step>
</Steps>
