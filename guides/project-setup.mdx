## Overview

FSP projects are created and managed through one of two toolchains: **e² studio** (Renesas' Eclipse-based IDE) or the **RA Smart Configurator (RASC)** used alongside IAR Embedded Workbench or Keil MDK. Both workflows use the same underlying FSP packs.

<Tabs>
  <Tab title="e² studio">
    ### New user setup

    If you are setting up FSP for the first time with e² studio, use the combined installer.

    <Steps>
      <Step title="Download the installer">
        Go to the [FSP v6.4.0 release page](https://github.com/renesas/fsp/releases/tag/v6.4.0) and download the **FSP with e² studio Installer** from the Assets section.
      </Step>
      <Step title="Run the installer">
        Run the installer executable. It installs:
        - e² studio IDE
        - FSP packs for the selected FSP version
        - GCC and/or LLVM toolchains
        - All other required tools

        No additional installations are required after the installer completes.
      </Step>
      <Step title="Create a new project">
        1. Open e² studio.
        2. Click **File > New > C/C++ Project**.
        3. In the new project window, select **Renesas RA** in the left pane.
        4. Follow the project wizard to select your target MCU, RTOS, and FSP version.
      </Step>
      <Step title="Configure your project">
        After the project is created, the RA Smart Configurator opens automatically inside e² studio. Use it to configure peripherals, assign pins, and add middleware stacks. See [RA Smart Configurator](/guides/configurator) for details.
      </Step>
    </Steps>

    ### Installing FSP packs only (existing users)

    If you already have e² studio installed and only need to update or add FSP packs, download the pack-only archive instead of the full installer.

    <Steps>
      <Step title="Download the pack archive">
        From the [FSP release page](https://github.com/renesas/fsp/releases), under the **Assets** section for your target version, download one of:

        - **`FSP_Packs_<version>.zip`** — works on any supported OS (Windows, macOS, Linux).
        - **`FSP_Packs_<version>.exe`** — self-extracting installer for Windows only.
      </Step>
      <Step title="Locate the e² studio support directory">
        The packs must be placed in the e² studio support area for your installation.

        To find the correct path:
        1. Open e² studio.
        2. Click **Help > About e² studio**.
        3. Click **Installation Details**, then select the **Support Folders** tab.
        4. The support area path is shown there.

        The path is typically:

        ```text
        ~/.eclipse/com.renesas.platform_<unique_number>
        ```

        <Note>
          e² studio must have been run at least once before this directory is created. If you have multiple e² studio installations, each has its own uniquely numbered support directory.
        </Note>
      </Step>
      <Step title="Extract the packs">
        - For the `.zip` file: extract its contents directly into the e² studio support directory.
        - For the `.exe` file: run it and it will extract to the correct location automatically.
      </Step>
      <Step title="Restart e² studio">
        Restart e² studio. The new FSP pack version will appear in the project wizard and configurator.
      </Step>
    </Steps>
  </Tab>

  <Tab title="RASC with IAR/Keil">
    ### Setup with IAR Embedded Workbench or Keil MDK

    If you use IAR Embedded Workbench or Keil MDK as your compiler and IDE, use the standalone RA Smart Configurator (RASC) for FSP project configuration.

    <Steps>
      <Step title="Install your toolchain">
        Install your preferred toolchain:
        - [IAR Embedded Workbench for Arm](https://www.iar.com/ewarm)
        - [Keil MDK](https://www.keil.com/mdk5)
      </Step>
      <Step title="Install RASC">
        Download and install the standalone RASC tool from the [FSP release page](https://github.com/renesas/fsp/releases). RASC handles FSP configuration and code generation independently of your compiler IDE.
      </Step>
      <Step title="Follow the RASC User Guide">
        For complete step-by-step instructions on creating and configuring a project with RASC alongside IAR or Keil MDK, see the official [RASC User Guide for MDK and IAR](https://renesas.github.io/fsp/_s_t_a_r_t__d_e_v.html#RASC-MDK-IAR-user-guide).
      </Step>
    </Steps>

    <Note>
      The RASC standalone tool and e² studio's integrated configurator share the same FSP pack format. Pack installation for standalone RASC follows the same directory conventions described in the e² studio tab.
    </Note>
  </Tab>
</Tabs>

## Version requirements

FSP and e² studio versions are paired. Each FSP release requires a minimum e² studio version. Using an older e² studio with a newer FSP pack is not supported.
