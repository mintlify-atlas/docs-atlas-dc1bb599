FSP v6.4.0 supports the full Renesas RA MCU product family, spanning five series that address use cases from ultra-low-power IoT nodes to high-performance industrial and graphics applications. All kits listed below work with the [FSP v6.4.0 installer](https://github.com/renesas/fsp/releases/tag/v6.4.0).

<Note>
  **RSSK and VK kits** — Renesas Starter Kit (RSSK) boards are compact, cost-optimized evaluation kits designed for quick prototyping. The VK-RA8M1 (Versatile Kit) provides an expanded feature set for evaluating advanced RA8 capabilities. Both kit types support the full FSP toolchain and RASC configurator.
</Note>

## Kit type reference

| Prefix | Full name | Description |
|--------|-----------|-------------|
| EK | Evaluation Kit | Full-featured evaluation board with headers, on-board J-Link, and peripheral connections |
| FPB | Fast Prototyping Board | Compact, low-cost board for rapid peripheral evaluation |
| MCK | Motor Control Kit | Includes inverter interface and connectors for motor drive evaluation |
| CK | Cloud Kit | Preconfigured for cloud connectivity evaluation (Wi-Fi, cellular, Ethernet) |
| AIK | AI Kit | Includes hardware for neural network and machine learning evaluation |
| RSSK | Renesas Starter Kit | Compact, cost-optimized starter board |
| VK | Versatile Kit | Expanded evaluation board with broad peripheral coverage |
| BGK | Board Game Kit | Specialized kit for interactive demonstration applications |
| VOICE | Voice Kit | Includes microphone and audio hardware for voice interface evaluation |
